=====================
Introduction to aiompesa
========================
`aiompesa` is an async wrapper for the `Safaricom Daraja API <https://www.daraja.safaricom.co.ke/>`_.

| `aiompesa` requires python version 3.6+

`aiompesa` is still under development.

Installation
=================
``pip install aiompesa``
